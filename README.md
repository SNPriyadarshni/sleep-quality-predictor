Sleep Quality Predictor
A Machine Learning model to predict sleep quality (Poor/Good) based on lifestyle factors like sleep duration, stress, BMI, steps, and activity levels. Built with Python and Scikit-learn, achieving 95%+ accuracy.​

Features
Exploratory Data Analysis (EDA) with correlation heatmaps and feature distributions.

Feature engineering: Encoding categoricals, scaling numerics.

Model: Extra Trees Classifier or Gradient Boosting for high precision/recall.

Evaluation: Confusion matrix, ROC-AUC, cross-validation scores.

Deployable as Streamlit/Flask app for real-time predictions.​

Tech Stack
Languages: Python 3.x

Libraries: Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

Dataset: Sleep health/lifestyle data
